# UC-001 日志文件读取与过滤

## 目标

用户可以指定一个本地日志文件，并按日志级别过滤输出。

## 命令示例

```powershell
go run ./cmd/log-tool --file ./testdata/sample.log --level ERROR
```

## 异常处理

- 日志文件不存在时提示“文件路径无效，请检查路径后重试”。
- 日志行格式错误时，工具在标准错误中输出诊断、将诊断写入系统临时告警文件，并继续处理后续日志。
- 临时告警文件仅在发现格式错误的日志行时创建；同一次运行的全部异常行写入同一个文件。
- 命令结束前会在标准错误中显示临时告警文件的绝对路径。
